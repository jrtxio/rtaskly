# Changelog

## [Unreleased]

## [0.0.3] - 2026-01-05
### Added
- 修复了macOS ARM64构建问题
- 改进了GitHub Actions工作流，添加了环境信息输出
- 添加了构建产物验证步骤
- 更新了构建脚本，确保在所有架构上正常工作

## [0.0.2] - 2026-01-04
### Added
- 支持多架构构建（x86_64和arm64）
- 改进了GitHub Actions工作流，为每个操作系统构建两个架构版本
- 更新了产物命名规则，包含版本、操作系统和架构信息

## [0.0.1] - 2026-01-04
### Added
- 初始版本发布
- 支持Windows、macOS和Linux平台
- 完整的测试套件
- 自动化构建和发布流程
- 切换到MIT许可证
- 为所有平台添加了应用图标支持
- 改进了窗口图标显示，解决了标题栏图标显示不完整的问题
- 优化了构建脚本，确保所有平台都生成纯图形界面应用
- 为Windows构建添加了图标嵌入支持
- 添加了多种尺寸的图标文件，支持不同分辨率显示
